A little literature...

Data structures and algorithms are two concepts that complement each other when it comes to writing clean, fast, concise, correct and efficient software, and code in general. When using these concepts, it is important to first define and fully layout the problem to enable the programmer choose appropriate Data Structures and Algorithms for the task at hand.

Main Applications

Data Structures are the programmatic way of storing data so that it can be retrieved and used efficiently. Every mature software project uses various types of data structures in one way or the other. Data Structures are therefore about rendering data elements in terms of some relationship, for better organization and storage.
