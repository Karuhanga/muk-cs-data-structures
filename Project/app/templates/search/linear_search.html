Linear Search

Linear search is a straight foward intuitive search procedure where one starts from the beginning of the list, continually searching throught the list in order until the search term is found.

Main Applications

Easy to implement

Advantages

Performance linearly proportional to the list size, which is a mark of a good algorithm.

Drawbacks

Very slow, compared to binary search, especially in cases where the target is towards the end of the list
