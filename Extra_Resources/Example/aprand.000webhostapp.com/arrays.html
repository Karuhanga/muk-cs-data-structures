Arrays Page

Arrays are a basic data structure that's the foundation of the most of the other data structures such as lists, stacks that are implemented through arrays.

An array is sequencial data storage cells all under a single variable name and ususally contain the same data type. we say usually because some languages like python can store various data types in the same array such as integers and characters whereas most languages like c and java store a single data type for a single array.

Arrays are important for; first of all,like we said, they are the basis of other data structures like lists and linked lists, they ease mathematical operations that are to be done on a large set of data e.g adding up the contents of an array and before file structures come into play, arrays are the basic way to store data before or after processing.

Arrays can be dynmaic or static depending on the language in question. static means the size of the array is constant through out the life time of the programme whereas dynamic the size of the array changes depending on the amount of data the array is required to hold. the c language to be particular has static arrays whereas most scripting languages like python have dynamic arrays.
