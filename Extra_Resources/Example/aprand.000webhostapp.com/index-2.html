Data Structures and Algorithms

Data structures are used to simplify handling(processing) large amounts of data. Algorithms are ideas of how data strutures should be implemented.

an algorithm is a step by step procedure, which defines a set of instructions to be executed in certain order to get the desired output. Algorithms are generally created independent of underlying languages.

From data structure point of view, following are some important categories of algorithms

  • Search : Algorithm to search an item in a datastructure.
  • Sort : Algorithm to sort items in certain order
  • Insert : Algorithm to insert item in a datastructure
  • Update : Algorithm to update an existing item in a data structure
  • Delete : Algorithm to delete an existing item from a data structure

Data structures are used in several disciplines such as operating systems(OS), compilers and database Management systems, data communications. Algorithms together with data structures are used in several applications. Common applications are: image processing, digital signal processing, simulation, numerical computations, cryptography, data compressions and genetic studies.
